-----

## 사용 방법

1. **헤더 입력** — Date, Clinic, WMC Name 입력 (날짜는 자동 입력)
1. **항목 체크** — 각 항목마다 **OPEN / CLOSED / N/A** 탭
- OPEN/CLOSED 선택 시 Comments 입력란 표시
- CLOSED 선택 시 Date Closed 입력란 추가 표시
1. **섹션 일괄 처리** — 섹션 헤더 오른쪽 버튼으로 한 번에 처리
- `✔ All CLOSED` — 섹션 전체 CLOSED
- `— All N/A` — 섹션 전체 N/A (약국 없는 클리닉, X-ray 없는 클리닉 등)
- `↺` — 섹션 전체 초기화
1. **섹션 이동** — 화면 오른쪽 **≡ NAV** 버튼으로 섹션 간 빠른 이동
1. **프린트** — 모든 항목 체크 완료 후 **🖨 PRINT** 버튼 활성화
- 미완료 항목 있으면 목록 표시 및 해당 위치로 이동
